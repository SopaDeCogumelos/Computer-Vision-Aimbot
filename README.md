# Computer Vision Aimbot
- Implementação e otimização de performance da arquitetura de redes neurais YOLO (You Only Look Once).
- Utilização de computação paralela (multithreading) para redução de latência no processamento de quadros.
- Desenvolvimento de algoritmos de controle com suavização de trajetória (motion smoothing) para simulação de comportamento humano e atuação em periféricos.
- Aplicação de heurísticas espaciais e segmentação semântica para priorização de alvos.
- Refinamento da detecção para mitigação de falsos positivos em ambientes com alta poluição visual.
